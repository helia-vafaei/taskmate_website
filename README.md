# To-Do App

A simple and user-friendly web application for managing daily tasks. This application allows users to create, update, complete, and delete tasks to stay organized and improve productivity.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Change background color
* Persistent task storage
